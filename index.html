How Our BackEnd works :

Fetching Stock Data: It uses the yfinance library to download historical stock data for the specified company (companyName).
Calculating Returns: It calculates the logarithmic returns of the adjusted closing prices of the stock.
Simulating Stock Prices: It then generates multiple simulations of future stock prices using a Monte Carlo simulation. For each simulation, it generates random returns based on the mean (mu) and standard deviation (sigma) of the historical returns. It then calculates the simulated stock prices over a specified number of days (252 days in this case).
Plotting Simulations: The code uses matplotlib.pyplot to create a plot with each simulation represented by a line. The final adjusted closing price of the actual stock data is marked with a horizontal line (ax.axhline).
Saving Plot to BytesIO Object: The plot is saved as a PNG image in a BytesIO object (img_buf). BytesIO is like a file in memory. Closing Plot: The plot is closed to free up memory.
Returning Image Data: The BytesIO object containing the plot image is returned. This image data can be used to display the plot on a web page or in any other application.
